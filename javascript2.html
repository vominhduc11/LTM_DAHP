Biến và Kiểu Dữ Liệu trong JavaScript

Trong JavaScript, các biến và kiểu dữ liệu là những khái niệm cơ bản và rất quan trọng. Việc hiểu rõ về biến và kiểu dữ liệu sẽ giúp bạn lập trình hiệu quả và tránh được các lỗi không mong muốn trong quá trình phát triển ứng dụng.

1. Biến trong JavaScript

Biến trong JavaScript là những đại diện cho dữ liệu mà bạn có thể lưu trữ và thay đổi trong suốt quá trình thực thi chương trình. Có ba cách để khai báo biến trong JavaScript:

  • var: Đây là cách khai báo biến truyền thống, nhưng hiện nay ít được sử dụng vì có một số hạn chế về phạm vi.
  • let: Được sử dụng thay cho var, có phạm vi trong block và giúp tránh những vấn đề liên quan đến phạm vi.
  • const: Dùng để khai báo các biến không thể thay đổi giá trị sau khi đã được gán.

Ví dụ về khai báo biến:


        let name = "John";
        const age = 25;
        var city = "Hanoi";

2. Kiểu Dữ Liệu trong JavaScript

JavaScript hỗ trợ một số kiểu dữ liệu cơ bản mà bạn có thể sử dụng để lưu trữ thông tin:

  • String (Chuỗi): Dùng để lưu trữ văn bản. Ví dụ: "Hello World"
  • Number (Số): Dùng để lưu trữ các giá trị số học. Ví dụ: 25, 3.14
  • Boolean (Kiểu logic): Dùng để lưu trữ các giá trị đúng/sai (true/false).
  • Object (Đối tượng): Dùng để lưu trữ các cặp key-value, là kiểu dữ liệu phức tạp.
  • Array (Mảng): Dùng để lưu trữ danh sách các giá trị. Ví dụ: [1, 2, 3, 4]
  • Null: Một kiểu đặc biệt chỉ có một giá trị duy nhất là null, dùng để chỉ sự không tồn tại.
  • Undefined: Kiểu này chỉ giá trị không được xác định hoặc biến chưa được gán giá trị.
  • Symbol (Biểu tượng): Một kiểu dữ liệu đặc biệt thường dùng trong lập trình phức tạp.

Ví dụ về các kiểu dữ liệu:


        let name = "Alice"; // String
        let age = 30; // Number
        let isActive = true; // Boolean
        let user = { name: "Alice", age: 30 }; // Object
        let numbers = [1, 2, 3, 4]; // Array
        let result = null; // Null
        let undefinedValue; // Undefined

3. Chuyển đổi kiểu dữ liệu (Type Conversion)

Trong JavaScript, đôi khi bạn cần phải chuyển đổi giữa các kiểu dữ liệu. JavaScript cung cấp các phương thức như String(), Number(), Boolean() để thực hiện việc chuyển đổi này.

Ví dụ về chuyển đổi kiểu dữ liệu:


        let num = "123";
        let convertedNumber = Number(num); // Chuyển từ String sang Number

        let bool = 1;
        let convertedBoolean = Boolean(bool); // Chuyển từ Number sang Boolean

4. Kết luận

Hiểu rõ về biến và kiểu dữ liệu trong JavaScript sẽ giúp bạn dễ dàng xử lý dữ liệu và tránh được các lỗi phổ biến khi lập trình. Hãy thử tạo các biến và sử dụng các kiểu dữ liệu trong JavaScript để làm quen và cải thiện kỹ năng lập trình của bạn!
