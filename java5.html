Cấu Trúc Điều Khiển trong Java

Cấu trúc điều khiển trong Java giúp điều khiển luồng thực thi của chương trình. Các cấu trúc này bao gồm các câu lệnh điều kiện, vòng lặp, và câu lệnh nhảy. Cấu trúc điều khiển cho phép chương trình đưa ra quyết định, lặp lại hành động hoặc thoát khỏi một phần mã trong quá trình thực thi.

1. Câu lệnh Điều kiện (if, else, else if)

Câu lệnh điều kiện được sử dụng để kiểm tra một điều kiện và thực hiện hành động tương ứng. Trong Java, cấu trúc điều kiện bao gồm if, else if, và else.

  • Câu lệnh if: Kiểm tra một điều kiện và thực thi mã nếu điều kiện đúng.
    • if (condition) {
    // Code to be executed if condition is true
}
  • Câu lệnh else: Thực thi mã nếu điều kiện trong if không đúng.
    • if (condition) {
    // Code to be executed if condition is true
} else {
    // Code to be executed if condition is false
}
  • Câu lệnh else if: Kiểm tra một điều kiện bổ sung khi điều kiện trong if không đúng.
    • if (condition1) {
    // Code if condition1 is true
} else if (condition2) {
    // Code if condition2 is true
} else {
    // Code if neither condition1 nor condition2 is true
}

2. Vòng Lặp (for, while, do-while)

Vòng lặp trong Java cho phép bạn thực thi một khối mã nhiều lần. Java hỗ trợ ba loại vòng lặp cơ bản:

  • Vòng lặp for: Được sử dụng khi số lần lặp là xác định trước.
    • for (initialization; condition; update) {
    // Code to be executed
}
  • Vòng lặp while: Được sử dụng khi điều kiện lặp được kiểm tra trước mỗi lần lặp.
    • while (condition) {
    // Code to be executed
}
  • Vòng lặp do-while: Được sử dụng khi cần đảm bảo ít nhất một lần lặp, vì điều kiện được kiểm tra sau mỗi lần lặp.
    • do {
    // Code to be executed
} while (condition);

3. Câu Lệnh Nhảy (break, continue)

Câu lệnh nhảy giúp bạn thoát khỏi một vòng lặp hoặc bỏ qua một vòng lặp hiện tại và tiếp tục với vòng lặp tiếp theo.

  • Câu lệnh break: Thoát khỏi vòng lặp hoặc câu lệnh switch.
    • for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break; // Exit the loop when i is 5
    }
    System.out.println(i);
}
  • Câu lệnh continue: Bỏ qua phần còn lại của vòng lặp và tiếp tục với lần lặp tiếp theo. 
    • for (int i = 0; i < 10; i++) {
    if (i == 5) {
        continue; // Skip the rest of the loop when i is 5
    }
    System.out.println(i);
}

4. Ví Dụ về Cấu Trúc Điều Khiển

Dưới đây là một ví dụ đơn giản sử dụng các cấu trúc điều khiển trong Java:


public class ControlStructures {
    public static void main(String[] args) {
        int number = 10;

        // Câu lệnh if-else
        if (number % 2 == 0) {
            System.out.println(number + " là số chẵn.");
        } else {
            System.out.println(number + " là số lẻ.");
        }

        // Vòng lặp for
        for (int i = 1; i <= 5; i++) {
            System.out.println("Số hiện tại: " + i);
        }

        // Câu lệnh break trong vòng lặp
        for (int i = 0; i < 10; i++) {
            if (i == 5) {
                break; // Thoát vòng lặp khi i = 5
            }
            System.out.println("Số hiện tại: " + i);
        }
    }
}
