Các Cấu Trúc Điều Khiển trong JavaScript

Cấu trúc điều khiển trong JavaScript giúp bạn kiểm soát dòng chảy của chương trình, cho phép thực hiện các hành động tùy thuộc vào các điều kiện khác nhau hoặc lặp lại một hành động nhiều lần.

1. Câu Lệnh Điều Kiện

Câu lệnh điều kiện cho phép thực thi một khối mã tùy thuộc vào điều kiện nhất định.

if

Câu lệnh if là cơ bản nhất để kiểm tra điều kiện:


if (condition) {
    // Thực thi mã khi điều kiện đúng
}

if...else

Câu lệnh if...else giúp xử lý trường hợp khi điều kiện không đúng:


if (condition) {
    // Thực thi mã khi điều kiện đúng
} else {
    // Thực thi mã khi điều kiện sai
}

if...else if...else

Câu lệnh if...else if...else dùng để kiểm tra nhiều điều kiện khác nhau:


if (condition1) {
    // Thực thi mã khi điều kiện 1 đúng
} else if (condition2) {
    // Thực thi mã khi điều kiện 2 đúng
} else {
    // Thực thi mã khi tất cả các điều kiện sai
}

2. Câu Lệnh Switch

Câu lệnh switch cho phép kiểm tra nhiều giá trị khác nhau của một biến, giúp tránh việc sử dụng nhiều câu lệnh if...else if.


switch (expression) {
    case value1:
        // Thực thi khi expression === value1
        break;
    case value2:
        // Thực thi khi expression === value2
        break;
    default:
        // Thực thi khi không có trường hợp nào khớp
}

3. Cấu Trúc Lặp

Cấu trúc lặp cho phép bạn thực thi một đoạn mã nhiều lần dựa trên một điều kiện nhất định.

for

Câu lệnh for dùng để lặp qua một khoảng thời gian xác định:


for (let i = 0; i < 5; i++) {
    // Thực thi mã lặp lại
}

while

Câu lệnh while lặp cho đến khi điều kiện không còn đúng:


while (condition) {
    // Thực thi mã khi điều kiện đúng
}

do...while

Câu lệnh do...while đảm bảo rằng mã trong khối lặp ít nhất sẽ được thực thi một lần:


do {
    // Thực thi mã ít nhất một lần
} while (condition);

4. Câu Lệnh Break và Continue

Câu lệnh break dùng để thoát khỏi một vòng lặp, còn continue dùng để bỏ qua một vòng lặp và tiếp tục với vòng lặp tiếp theo.

Ví dụ với break:


for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break; // Thoát khỏi vòng lặp khi i bằng 5
    }
}

Ví dụ với continue:


for (let i = 0; i < 10; i++) {
    if (i === 5) {
        continue; // Bỏ qua khi i bằng 5
    }
    console.log(i);
}

5. Kết luận

Cấu trúc điều khiển trong JavaScript là công cụ mạnh mẽ giúp bạn kiểm soát luồng chương trình, xử lý các điều kiện và lặp lại các hành động. Việc nắm vững các cấu trúc này là rất quan trọng để phát triển các ứng dụng hiệu quả và linh hoạt.
